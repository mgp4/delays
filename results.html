Závěr pro nás

Testovací běh ukázal, že některá ze vstupních bariér, ať už jde o nutnout používání ssh nebo linuxového serveru je příliš velká. Příští běh (na větších a bohatších datech) tak nejspíš zjednodušíme: podobně jako v soutěží Kaggle budou účastníci odevzdávat jen dataset. Na samotné analýzy pak budou moct používat hardware a software který jim vyhovuje nejvíc.

Update

 

Predikované výstupy jsme ověřovali proti tomuto testovacímu datasetu.

import csv
import sys
import arrow

team_file = sys.argv[1]
real_file = "delays_real.csv"
counter = 0
def get_diff(a,b):
        prdt_time = arrow.get(a, 'YYYY-MM-DD HH:mm:ss')
        real_time = arrow.get(b, 'YYYY-MM-DD HH:mm:ss')
        later = max(prdt_time, real_time)
        sooner = min(prdt_time, real_time)
        return (later - sooner).total_seconds()
with open(team_file) as tf, open(real_file) as rf:
        sumdiff = 0
        for rl in rf:
                tl = tf.readline()
                counter += 1
                if counter == 1:
                        continue
                tl = tl.strip()
                rl = rl.strip()
                tcells = tl.split(",")
                rcells = rl.split(",")

                scheduled_time = rcells[5].strip()
                real_time = rcells[6].strip()
                if rcells[:6] == tcells[:6] and tcells[6].strip():
                        prdt_time = tcells[6].strip()
                        try:
                                sumdiff += get_diff(prdt_time, real_time)
                        except:
                                sumdiff += get_diff(scheduled_time, real_time) # fallback
                else:
                        sumdiff += get_diff(scheduled_time, real_time) # fallback
print("teamfile: {} difference: {}".format(team_file, sumdiff))
